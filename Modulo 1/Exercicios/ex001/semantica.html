O que é semântica?

É significado. É mais relevante o significado do que a forma.O HTML5 está focado apenas no significado. Qualquer personalização deve ser feita no CSS3. Os mecânismos de busca podem diminuir o engajamento do site se estiver usando tags obsoletas de personalização no HTML.
